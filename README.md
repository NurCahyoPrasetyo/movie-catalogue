## Installation

Install my-project with npm

```bash
  clone my-project
  cd my-project
  npm install
  npm run start-dev
```

## Deployment

To deploy this project run

```bash
  npm run build
```

## Running Tests

To run tests, run the following command

```bash
  npm run test
```
